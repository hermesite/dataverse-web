FEATURES

No coding required

Dataverse works with Google Sheets as 'Backend Data'. That means that you can completely define the interactive experience by just filling out a Google Sheet and passing its shared URL to the application. You can read more about the specifics of the Google Sheets format here.

Multiheadset

Dataverse is built on AFrame, which in turn is based on WebVR standards. That means that you can enjoy the generated experience in WebVR compatible devices accessing the exact same URL in a web browser. Dataverse has been tested on Google Cardboard (Chrome browser), HTC Vive and Oculus Rift (Firefox browser) and Oculus Go.

Open Source

Since Dataverse is Open Source, you can code new visualization types, improve existing features or create new ones.
